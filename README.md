# Jan Seva Mitra

Jan Seva Mitra is a smart complaint analysis and public guidance system designed to help citizens understand public problems, identify the responsible department, estimate priority, and get useful welfare guidance.

## Features

- Complaint category prediction using Machine Learning
- Department suggestion
- Priority level detection
- Problem understanding and solution steps
- Scheme recommendation
- Public awareness page
- Hindi + English support
- Streamlit-based user interface

## Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- NLTK
- Matplotlib

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
