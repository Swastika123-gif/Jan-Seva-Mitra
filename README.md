# 🇮🇳 Jan Seva Mitra

### Smart Public Complaint & Citizen Guidance System

---

## 🌍 About the Project

**Jan Seva Mitra** is a **machine learning-based public complaint and guidance system** designed to help citizens understand their problems and take appropriate action.

The system analyzes user complaints, identifies the relevant department, estimates urgency, and provides step-by-step guidance along with information about government support schemes.

It is built with a focus on **real-world usability, accessibility, and citizen awareness**, especially for rural and underprivileged communities.

> 💡 *Empowering citizens through awareness, guidance, and technology.*

---

## 🎯 Problem We Are Solving

Many people:

* ❌ Don’t know which department to contact
* ❌ Don’t understand their rights
* ❌ Lack awareness about government schemes
* ❌ Face delays due to confusion

👉 Our solution bridges this gap using **Simple UI**

## 🤖 Machine Learning Models

| Model               | Role                             |
| ------------------- | -------------------------------- |
| Logistic Regression | Baseline                         |
| Naive Bayes         | Fast prediction                  |
| Linear SVM          | ⭐ Best accuracy (used in system) |
| Decision Tree       | Rule-based                       |
| Random Forest       | Ensemble learning                |

---

## 📊 Model Performance

* Linear SVM achieved **highest accuracy (~95–100%)**
* Selected for final deployment
